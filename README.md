#  Social Media Post Composer

A modern React.js application for creating social media posts with live character count validation for multiple platforms.

##  Features

-  Twitter (280 characters)
-  Facebook (63206 characters)
-  Instagram (2200 characters)
-  LinkedIn (3000 characters)
-  Live character counter
-  Progress bar
-  Character limit validation
-  Disable Publish button for invalid posts
-  Modern responsive UI

##  Tech Stack

- React.js
- JavaScript (ES6)
- HTML (JSX)
- CSS3
- Node.js
- npm
